Favicon Instructions

For production use, you should place a favicon.ico file in the public directory.

You can create a favicon using various online tools or image editors:

1. Create a square image (recommended size: 32x32 or 64x64 pixels)
2. Save it as favicon.ico
3. Place it in this directory (public/)

The favicon will be displayed in browser tabs and bookmarks.

Recommended favicon for this application:
- A simple icon representing movies or TV shows
- A film reel, clapperboard, or TV icon
- Use colors that match the application's theme (blue and dark blue)

Online tools for creating favicons:
- https://favicon.io/
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/
